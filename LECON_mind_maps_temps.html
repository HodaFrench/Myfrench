🗺️ Mind Maps — زمان‌های فرانسه

هر زمان در یک نگاه: فرمول + کاربرد + مثال + مقایسه. از A1 تا B2.
🔵
Présent de l'indicatif
زمان حال
Radical + -e/-es/-e/-ons/-ez/-ent (groupe 1)
Radical + -is/-is/-it/-issons/-issez/-issent (groupe 2)
📌 کاربرد اصلی
✓ فعل و حالت فعلی
✓ حقایق و عادت‌ها
✓ برنامهٔ آینده (= futur proche)
💬 مثال
Je travaille tous les jours.
Il fait du sport le matin.
Demain, je pars à Paris.
هر روز کار می‌کنم · صبح‌ها ورزش می‌کند
نکته: با «maintenant، en ce moment، d'habitude، toujours، souvent» → Présent
🟤
Passé Composé
گذشته‌ی ترکیبی
avoir / être (présent) + participe passé
📌 کاربرد
✓ رویداد مشخص در گذشته
✓ عملی که تمام شده
✓ تعداد مشخص بار
💬 مثال
J'ai mangé une pizza.
Elle est partie hier.
Nous avons vu ce film.
یه پیتزا خوردم · دیروز رفت
با être: aller, venir, partir, arriver, naître, mourir, rester, entrer, sortir + تمام انعکاسی‌ها
توافق: با être: participe با فاعل توافق می‌کند. J'ai / Elle est partie.
🟣
Imparfait
گذشتهٔ استمراری
nous (présent) → radical + -ais/-ais/-ait/-ions/-iez/-aient
📌 کاربرد
✓ عادت در گذشته
✓ توصیف زمینه
✓ حالت مستمر در گذشته
✓ بعد از «Si» برای شرط
💬 مثال
Quand j'étais petit, je jouais.
Il pleuvait et le vent soufflait.
Si j'avais le temps...
وقتی بچه بودم بازی می‌کردم · باران می‌بارید
Passé composé vs Imparfait: PC = عمل کوتاه و مشخص · Imparfait = زمینه و استمرار
«Il lisait quand le téléphone a sonné.» (داشت می‌خواند که زنگ زد)
🟢
Futur Simple
آینده‌ی ساده
Infinitif + -ai/-as/-a/-ons/-ez/-ont
📌 کاربرد
✓ آینده‌ی دور
✓ وعده و تعهد
✓ بعد از «quand/lorsque» → futur (مهم!)
💬 مثال
L'année prochaine, je voyagerai.
Je te téléphonerai demain.
Quand tu arriveras, appelle.
سال دیگر سفر می‌کنم · وقتی رسیدی زنگ بزن
بی‌قاعده‌ها: être→ser· avoir→aur· aller→ir· faire→fer· venir→viendr· voir→verr
🔴
Conditionnel Présent
شرطی حال
Infinitif + -ais/-ais/-ait/-ions/-iez/-aient
(همان پایهٔ futur + پسوندهای imparfait)
📌 کاربرد
✓ شرط غیرواقعی (Si + imparfait)
✓ درخواست مؤدبانه
✓ توصیه (devrais)
✓ آرزو (j'aimerais)
✓ خبر غیرتأییدشده
💬 مثال
Si j'avais de l'argent, je voyagerais.
Je voudrais un café.
Tu devrais dormir.
Il serait malade. (گفته می‌شود)
⚠️ هیچ‌وقت conditionnel بعد از «Si» نمی‌آید!
✅ Si j'avais (imparfait) → je ferais (conditionnel)
🟠
Subjonctif Présent
التزامی حال
ils (présent) → radical + -e/-es/-e/-ions/-iez/-ent
📌 کاربرد
✓ بعد از «que» + احساس
✓ بعد از il faut que
✓ بعد از bien que / quoique
✓ بعد از pour que / afin que
✓ بعد از avant que
💬 مثال
Il faut que tu viennes.
Je veux qu'il parte.
Bien qu'elle soit fatiguée...
Pour que tu comprennes.
باید بیایی · می‌خواهم برود
بی‌قاعده‌های مهم: être→sois/soit/soyons · avoir→aie/ait/ayons · aller→aille · faire→fasse · pouvoir→puisse · vouloir→veuille
⚖️
مقایسهٔ زمان‌ها — یه نگاه
زمان سیگنال‌های اصلی مثال
Présentmaintenant, souvent, d'habitude, chaque jourJe mange.
Passé composéhier, l'année dernière, soudain, une foisJ'ai mangé.
Imparfaitquand j'étais..., toujours, autrefois, Si + condJe mangeais.
Futur simpledemain, bientôt, l'année prochaine, quand+futurJe mangerai.
ConditionnelSi+imparfait, je voudrais, on devrait, il seraitJe mangerais.
Subjonctifque+sentiment, il faut que, bien que, pour queque je mange
تفاوت PC vs Imparfait:
«Hier, il faisait (imparfait - زمینه) beau quand l'orage a éclaté (PC - رویداد).»
هوا خوب بود (imparfait = زمینه) که طوفان شروع شد (PC = رویداد کوتاه)
